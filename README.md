
Gui for corrSynth.
Purpose: integrate in STELAR KLMS as streamlit webapp.

Required python packages:
- streamlit
- pandas
- numpy
- stelar-client >= 0.3.6
- stelar-synth-corr-data >= 0.1.0

install with:
```bash
pip install streamlit pandas numpy stelar-client==0.3.6 stelar-synth-corr-data==0.1.0
```
